install reactjs via mise
-> Creeër een frontend folder en ga hierin via 'pwsh: cd frontend'
- npm create vite@latest [Naam,deed ik niet] --template diabeticum_pedis
-- note: diabeticum_pedis == diabetische voet, ik koos dit om issues met 'frontend' als naam te voorkomen
- framework: Vanilla
- Variant: Typescript
- Rolldown-vit: No

Zet mise op je project
-> creeër een file genaamd '.mise.toml' met het volgende erin
    [tools]
    node = "lts"
    npm = "latest"
- pwsh: mise install
- mise ls
-- mise ls controleert de installatie, je zou het volgende moeten kunnen zien
PS C:\[ROOT]\frontend> mise ls
Tool  Version  Source                                             Requested
node  24.11.1  C:\[ROOT]\frontend\.mise.toml  lts
npm   11.6.3   C:\[ROOT]\frontend\.mise.toml  latest
- pwsh: npm run dev   

Controleer mise volledige installatie - was van toepassing bij mij
- pwsh: mise doctor
